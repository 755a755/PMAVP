# PMAVP
PMAVP
Antiviral peptides (AVPs) are promising candidates for antiviral drug development, and accurate computational prediction of AVPs can accelerate peptide screening and reduce experimental costs. However, existing methods still suffer from class imbalance, dependence on handcrafted features, limited modeling efficiency, insufficient interpretability, and insufficient consideration of functional activity prediction. To address these limitations, we propose PMAVP, a deep learning framework that integrates the ProtT5 protein pretrained language model with the Mamba architecture for both AVP identification and functional activity prediction. PMAVP can distinguish AVPs from non-AVPs and further predict their activities against eight viral species and six viral families. In this framework, ProtT5 is used to extract deep semantic representations from peptide sequences, while Mamba efficiently captures long-range dependencies with lower computational complexity than conventional self-attention-based models. Global average pooling and global max pooling are combined to integrate global sequence information and key local features. Focal Loss is introduced to alleviate class imbalance, and transfer learning is adopted to improve functional activity prediction. For viral species and family prediction, a multi-task learning strategy is further employed to jointly learn shared antiviral representations and task-specific discriminative features. Experimental results show that PMAVP achieves favorable performance in prediction accuracy, stability, and computational efficiency. Ablation analysis confirms the importance of the Mamba module and global max pooling. Moreover, DeepSHAP-based interpretability analysis suggests that the first 40 amino acids contribute substantially to AVP prediction. Overall, PMAVP provides an efficient and interpretable framework for AVP identification and functional activity prediction.

## File Description

The `Dataset` folder contains all sequences from the three datasets used in this study.

Before training the PMAVP model, please run `feature_extraction.py` to extract features using the ProtT5 pretrained protein language model. After feature extraction, run the main program `PT5first_stage_mamba.py`, which includes both the training and testing procedures for the PMAVP model.
